# GenerativeAIPracticalJavaBasics

A bite-sized Java playground that teaches you how to use OpenAI’s APIs in just a few files.

## What you’ll learn
- Make your first API call to an OpenAI model

## Prerequisites
- JDK 17 or higher
- Maven or Gradle (wrappers included)
- An OpenAI API key (`OPENAI_API_KEY` env var)
